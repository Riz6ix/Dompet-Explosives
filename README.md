## 🤔 Apa Sih Ini?

**Dompet Explosives** adalah web app buat ngurus kas kelas — dari A sampe Z. Kas mingguan, iuran, cicilan, transaksi, sampe laporan PDF yang tinggal print.

Yang bikin beda? Ada sistem **RPG** 🎮 — setiap aksi dapet XP, naik level, unlock achievement. Jadi ngurus duit kelas berasa main game.

```
🧑‍💼 Kamu                          📱 Dompet Explosives
  │                                    │
  ├── "Siapa yang belum bayar?"  ──►   ├── Cek Bayar ✅
  ├── "Total kas berapa?"        ──►   ├── Dashboard 📊
  ├── "Bikin laporan dong"       ──►   ├── PDF Export 📄
  ├── "Ada yang nyicil nih"      ──►   ├── Cicilan 💳
  └── "Bosen ngurus kas"         ──►   └── Quest & XP 🎮
```

> **💡 Satu kalimat:** Bendahara kelas tapi make it ✨ fun ✨

> **📱 Note:** App ini didesain **mobile-first** — tampilannya paling optimal di HP. Buka di desktop juga bisa, tapi experience terbaiknya di layar kecil.

## 🚀 Quick Start

### Yang dibutuhin

- [**Node.js**](https://nodejs.org/) versi 18+ (disarankan LTS)
- **npm** 9+ (udah include bareng Node.js)
- [**Git**](https://git-scm.com/)

### Cara install

```bash
# Clone repo ini
git clone https://github.com/Riz6ix/Dompet-Explosives.git

# Masuk folder project
cd Dompet-Explosives

# Install dependencies
npm install

# Jalanin dev server
npm run dev
```

Buka **http://localhost:5173** → ikutin setup wizard → done. ✅

### Command lainnya

| Command | Fungsi |
|---|---|
| `npm run dev` | Jalanin development server |
| `npm run build` | Build untuk production |
| `npm run preview` | Preview production build di lokal |
| `npm run lint` | Cek linting errors |

> [!NOTE]
> Project ini pake **Vite 8 beta** dan **Tailwind CSS 4**. Pastiin Node.js kamu minimal versi 18.

## 🔐 Data & Privacy

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│   📱 Browser kamu ◄──── Data tersimpan di sini      │
│                                                     │
│   ☁️  Server       ◄──── KOSONG. Gak ada apa-apa.   │
│                                                     │
│   👀 Yang bisa liat ◄──── Cuma kamu.                 │
│                                                     │
└─────────────────────────────────────────────────────┘
```

- **100% Client-Side** — Data cuma ada di browser kamu, gak kemana-mana
- **Zero Tracking** — Gak ada analytics, gak ada cookies, gak ada telemetry
- **PIN Lock** — Optional lock screen buat jaga dari tangan iseng
- **Backup JSON** — Export kapan aja, import di browser/device lain

> [!WARNING]
> Clear browser data = data hilang. **Selalu backup** sebelum clear cache atau ganti device!
